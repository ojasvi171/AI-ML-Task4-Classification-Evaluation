# AI & ML Task 4 - Classification Models & Evaluation Metrics

## Project Overview

This project focuses on building and evaluating classification models using the Breast Cancer Dataset. The objective is to classify tumors as malignant or benign while understanding classification metrics, ROC-AUC analysis, and techniques for handling imbalanced data.

## Dataset

Breast Cancer Dataset from Scikit-learn.

Target Classes:

* 0 = Malignant (Cancerous)
* 1 = Benign (Non-Cancerous)

The dataset contains medical features extracted from breast cancer cell images and is commonly used for binary classification tasks.

## Project Workflow

1. Data Loading
2. Data Exploration
3. Train-Test Split
4. Feature Scaling using StandardScaler
5. Logistic Regression Model
6. Confusion Matrix Analysis
7. Precision, Recall and F1-Score Evaluation
8. ROC Curve and AUC Score Analysis
9. Handling Imbalanced Data using Class Weights
10. Decision Tree Classifier
11. Model Comparison
12. Final Model Selection

## Models Used

* Logistic Regression
* Decision Tree Classifier

## Evaluation Metrics

* Accuracy

* F1-Score

## Results

| Model                    | Accuracy |
| ------------------------ | -------- |
| Logistic Regression      | 98.25%   |
| Decision Tree Classifier | 91.23%   |

AUC Score (Logistic Regression): 0.9954

The Logistic Regression model achieved the best overall performance and was selected as the final model.
